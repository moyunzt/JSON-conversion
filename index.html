JSON/YAML 转换工具

在JSON和YAML格式之间轻松转换，支持格式化、键排序和注释处理

使用说明

  1. 选择输入格式（JSON或YAML）
  2. 粘贴您的代码到输入框
  3. 选择输出格式
  4. 设置所需选项
  5. 点击"转换"按钮

关于

这个工具使用纯前端技术（HTML, CSS, JavaScript）构建，无需后端服务器。所有转换都在您的浏览器中本地完成，不会上传您的数据。

支持的格式：

  • JSON - JavaScript Object Notation
  • YAML - YAML Ain't Markup Language
  • TOML - Tom's Obvious, Minimal Language
  • XML - Extensible Markup Language
